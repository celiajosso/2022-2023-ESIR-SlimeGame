# Projet d'ouverture informatique

## Description

Ce projet comprend:

- un jeu de platforme en 2D réalisé avec pygame.
- un éditeur de niveau réalisé avec pygame.
- des diagrammes [UML](./uml/) pour mieux comprendre l'architecture du projet.
- un [rapport](https://docs.google.com/document/d/1jUlnGIDz0XbSrYPMJiN361MfTO0F69mT/edit?usp=sharing&ouid=105145370196956074795&rtpof=true&sd=true) expliquant les choix techniques du projet
